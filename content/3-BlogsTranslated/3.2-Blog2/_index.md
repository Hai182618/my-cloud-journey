# AWS Weekly Roundup — The Latest AWS Updates

Every Monday, we bring you a summary of the most important AWS announcements of the week. This edition highlights updates across **Amazon Bedrock, EC2 C8gn, DynamoDB Global Tables, Aurora DSQL, Amazon Nova Canvas**, and more.

---

## EC2 C8gn — Network Performance up to 200 Gbps

{{< figure src="/images/blog2/image1.png" title="Figure 1. EC2 C8gn powered by AWS Graviton4." >}}

EC2 C8gn instances are built on **AWS Graviton4** and the latest **Nitro System**, offering:

- Up to **200 Gbps network throughput**
- Significant compute performance gains over previous generations
- Optimized for workloads such as:
  - High-performance networking
  - Telecom and packet processing
  - Distributed data systems
  - Real-time analytics

---

## DynamoDB Global Tables — Multi-Region Strong Consistency

AWS introduces **Multi-Region Strong Consistency (MRSC)** for DynamoDB Global Tables, allowing:

- Strongly consistent reads from *any* replicated Region  
- Simplified development for global financial and mission-critical applications  
- Zero-RPO data protection guarantees  

---

## Amazon Nova Canvas — New Creative Styles

Amazon Nova Canvas now includes:

- **Eight new generation styles**  
- Enhanced **virtual try-on** experiences  
- Better outputs for creative and e-commerce use cases  

---

## Amazon Q in Connect — Now Supports More Languages

Amazon Q adds multilingual support across:

- English  
- Spanish  
- French  
- Portuguese  
- Mandarin  
- Japanese  
- Korean  

---

## Aurora MySQL & RDS MySQL — Integration with SageMaker

New integrations simplify ML pipelines:

- Near-real-time data extraction  
- Apache Iceberg table format support  
- Shared data access for analytics engines  

---

## Aurora DSQL — Now Available in Additional Regions

Aurora DSQL expands to:

- APAC (Seoul)
- New European Regions

Benefits include:

- Serverless distributed SQL  
- Global cluster awareness  
- Unlimited horizontal scalability
