### Week 1 Achievements:

* Gained a solid foundational understanding of Cloud Computing, including its core characteristics, service models (IaaS, PaaS, SaaS), and reasons companies migrate to AWS.

* Understood key differentiators of AWS: scalability, flexibility, global coverage, security posture, high availability, and rapid innovation.

* Learned how to begin a cloud adoption journey—assessment, migration planning, service selection, and the concept of landing zones.

* Mastered AWS Global Infrastructure components: Regions, Availability Zones, Edge Locations, Local Zones, and Wavelength Zones, and how they influence architectural decisions.

* Became proficient in AWS service management tools:  
  * AWS Management Console  
  * AWS CLI  
  * AWS CloudShell  
  * AWS Documentation  
  * AWS Pricing Calculator  

* Understood AWS cost optimization: instance selection, Auto Scaling, Reserved Instances, Savings Plans, and Cost Explorer monitoring.

* Learned how to work effectively with AWS Support: support plans, benefits, case creation, and issue tracking.

* Completed hands-on labs that reinforced Module 01 concepts through navigation, scenario exploration, and foundational cloud operations.
